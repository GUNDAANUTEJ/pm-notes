
# 2) ReadMe document

## 2 a) List of all pre-requisite software’s:

1. IntelliJ.
2. Maven
3. JDK 17
4. Java c compiler with Project bytecode version is 12 and Target bytecode version is 12 in the settings.

## 2 b) Instructions for running the project:
